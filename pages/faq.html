Perguntas Frequentes

Tire suas dúvidas sobre o EcoVolt

O que é o EcoVolt?

+

É uma solução gamificada para incentivar ações sustentáveis reais dentro da plataforma SoulUp.

Como o usuário ganha pontos?

+

Por meio de ações sustentáveis validadas, missões concluídas, login diário e interações relevantes.

Como uma ação é validada?

+

O usuário envia um vídeo. O sistema utiliza regras determinísticas, análise contextual, apoio de IA e revisão manual em casos específicos.

O anúncio substitui o login diário?

+

Não. O anúncio apenas recupera a sequência perdida, mas não cria um login válido artificial.

Como funcionam os tiers?

+

Os usuários são classificados em Ferro, Bronze, Prata, Ouro, Diamante e Sustentabilístico, com progressão mensal baseada no ranking.

Qual é a principal recompensa?

+

O primeiro colocado de cada tier pode receber subsídio na conta de energia, respeitando o teto definido pelo projeto.

O sistema evita fraude?

+

Sim. Há limites de repetição, redução de pontuação por ações repetidas e validação híbrida dos envios.

Existe limite de ações sustentáveis por dia?

+

Sim. Cada ação específica pode ser realizada até 3 vezes por dia para evitar uso indevido do sistema.

O que acontece se eu repetir a mesma ação muitos dias seguidos?

+

A pontuação pode ser reduzida progressivamente para incentivar variedade nas ações sustentáveis.

Posso contestar uma validação recusada?

+

Sim. O sistema permite solicitação de revisão em casos de erro ou inconsistência na validação.

Quantas missões posso receber por semana?

+

O usuário pode receber até 5 missões semanais, variando conforme nível, tier e comportamento.

Como funciona o ranking?

+

O ranking é atualizado ao longo do mês e considera o desempenho do usuário nas atividades do sistema.

Os pontos são resetados todo mês?

+

O ranking mensal é reiniciado, porém o progresso geral do usuário permanece salvo.

Como subir de tier?

+

A progressão acontece conforme a posição do usuário no ranking ao final do ciclo mensal.

Existe ranking entre amigos?

+

Sim. O EcoVolt possui ranking global e ranking entre amigos com filtros personalizados.

Meus vídeos ficam armazenados?

+

Os envios podem ser armazenados temporariamente para validação, auditoria e revisão quando necessário.

Quais tipos de ações sustentáveis posso enviar?

+

As ações são organizadas por categorias e ODS, envolvendo práticas ambientais e sociais sustentáveis.

Como funciona o sistema de níveis (LVL)?

+

Os usuários acumulam XP e evoluem até o nível máximo definido pelo sistema.

O que é necessário para validar o login diário?

+

O usuário precisa atingir o tempo mínimo de uso diário definido pela plataforma.
